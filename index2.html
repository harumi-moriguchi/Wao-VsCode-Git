************************************************************************ GitHub Pull Requests 拡張機能インストール ログインしなおす https://nuxt.com/docs/4.x/getting-started/prerendering npx nuxt generate npx serve .output/public ************************************************************************ 1. インストール プロジェクトのルートディレクトリで以下のコマンドを実行し、モジュールをインストールします。 # npmの場合 npm install -D @nuxtjs/tailwindcss 2. nuxt.config の設定 nuxt.config.ts ファイルの modules に、インストールしたモジュールを追加します。 // nuxt.config.ts export default defineNuxtConfig({ modules: [ '@nuxtjs/tailwindcss' ], compatibilityDate: '2024-04-03', // Nuxt 4を使用するための設定 }) ************************************************************************ 1. モジュールのインストール まず、プロジェクトのルートディレクトリで以下のコマンドを実行し、パッケージをインストールします。 bash npm install -D @nuxtjs/google-fonts 2. nuxt.config.ts の設定 nuxt.config.ts にモジュールを追加し、使いたいフォントファミリーを設定します。 typescript // nuxt.config.ts export default defineNuxtConfig({ modules: [ '@nuxtjs/google-fonts' ], googleFonts: { // 使用するフォントの設定 families: { 'Noto+Sans+JP': [400, 700], // 例：Noto Sans JPのRegularとBold Roboto: { wght: [100, 400, 700], ital: [100] // イタリック体も指定可能 }, }, // フォントをローカルにダウンロードして配信する場合 (おすすめ) download: true, // inject: true, // デフォルトでtrue: CSSに自動注入 } }) ************************************************************************ bash npm install @vesp/nuxt-fontawesome nuxt.config.ts にモジュールを追加し、使用するアイコンを指定します。 typescript // nuxt.config.ts export default defineNuxtConfig({ modules: [ '@vesp/nuxt-fontawesome' ], fontawesome: { icons: { solid: ['coffee', 'child', 'user', 'home'], regular: ['comment', 'calendar', 'clock'], brands: ['twitter', 'github', 'facebook'] } } }) ************************************************************************
